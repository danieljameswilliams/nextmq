# NextMQ Marketing Website

Marketing website for the NextMQ package, showcasing its features, installation, and usage.

## Development

```bash
npm run dev
```

The site will be available at `http://localhost:3001`.

## Build

```bash
npm run build
npm start
```

## Features

- Modern, responsive design with Tailwind CSS
- Dark mode support
- Code examples and documentation
- Installation and quick start guides
- Feature highlights
